1.Installation

  1. Close MetaTrader 5 (if open).
  2. Open the data folder: File → Open Data Folder.
  3. Copy Dralvo_GoldMaster.ex5 into MQL5\Experts\. (Or copy the .mq5 and Compile (F7).)
  4. CFTC data source — pick one: (a) live via API (recommended) → set UseLiveAPI=true + whitelist the URL (section 2), no CSV needed; (b) file mode → set UseLiveAPI=false and you must copy cftc_status.csv into MQL5\Files\ (missing it shows CFTC FILE NOT FOUND). Backtests always need the CSV.
  5. Reopen MT5 → Navigator → Expert Advisors, right-click Refresh.
  6. Open an XAUUSD chart, switch to the D1 timeframe, and drag Dralvo GoldMaster onto the chart.
  7. Common tab: tick Allow Algo Trading. Inputs tab: enter your License key + MT5 Account (section 2), load a .set.
  8. Click OK → enable Algo Trading on the toolbar (lights up green).
This is a daily system: signals are evaluated once per new D1 candle. It is normal to go several days between trades — don't expect intraday entries.

2.License activation — REQUIRED for demo/live

The EA has a license gate: on a real chart it calls the Dralvo server to verify the key. Without this step the EA will NOT run and shows the warning Contact telegram: @edgardinh86.

In the Strategy Tester (backtest) the license gate is skipped — backtests always run without a key.

Step 1 — Get your License key

How you get the key depends on your plan:

PlanWhere to get the License key
VIPSign in to the Dashboard at https://www.dralvo.com → License / Key → generate a key for each MT5 account.
FreeContact Telegram @edgardinh86 to receive a License key.
  • Whichever plan, the key is bound to a specific MT5 account number — you must provide/register the correct account number.
  • Find your account number: Navigator, or Tools → Options → Server, or the MT5 window title (6–10 digits).

🔹 VIP plan — out of account slots: each VIP account gets a set number of slots. Once you've used all (max) accounts and want to add a new one, contact the admin to be granted more.

Step 2 — Whitelist WebRequest ⚠️ Most-forgotten step

  1. Open Tools → Options (Ctrl+O).
  2. Go to the Expert Advisors tab.
  3. Tick ☑ Allow WebRequest for listed URL.
  4. Type exactly: https://www.dralvo.com
  5. Click OK.
The same URL is also used for the live CFTC API when UseLiveAPI=true. Without whitelisting → license fails and the dashboard shows API: OFFLINE. This is the #1 reason the EA won't run even with the key entered.

3.How this EA trades

On each new D1 candle, the EA only opens a BUY when all filters are met at once:

FilterMeaning
CFTC bullishCFTC positioning leans long. Source: live + UseLiveAPI=true → straight from the API; otherwise (backtest, or UseLiveAPI=false) → read from cftc_status.csv.
TRENDEMA50 > EMA200 on D1 (uptrend).
PB (Pullback)Price has retraced far enough from the recent high (PullbackPct, PullbackBars).
HL (Higher-Low)The most recent candle low is above the lowest low of ~20 candles ago (bullish structure).

It also needs a cooldown (≥ 5 days since the last trade, at most 1 trade per day). On entry: SL = price − 1.5×ATR, sizing by % risk. Management: trailing TrailATR×ATR from the high, forced close after MaxHoldDays.

The dashboard shows 3 tokens TREND / PB / HL (green = passed, red = not yet). The Experts/Journal tab prints an ENTRY-CHK … line each candle telling you exactly which gate blocked entry.

Load a set: Inputs → Load → pick the file for your appetite. Save your own config: Save.

Setfiles by account size (quick pick)

To match how the other EAs are selected, there is also a set of Dralvo_GoldMaster_<TIER>_<size>.set files (24 files = 4 tiers × 6 sizes: 1k/5k/10k/20k/50k/100k). Just pick the right appetite + balance and load it.

💡 Important: GoldMaster sizes lots entirely from RiskPercent (% of balance × SL distance) so it auto-scales to any account size. That's why files of the same tier (e.g. every BALANCED_*) are identical — the size in the filename is only for quick picking and doesn't change how the EA runs. One base tier file is enough for both 1,000 and 100,000 USD.
If using the CSV file (UseLiveAPI=false), it must be updated periodically so "CFTC bullish" reflects the present. To avoid manual updates → set UseLiveAPI=true so the EA fetches via the API. InpLicenseKey/InpMT5Account are per-user and are not stored in shared .set files.
