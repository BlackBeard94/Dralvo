1.Instalación

  1. Cierre MetaTrader 5 (si está abierto).
  2. Abra la carpeta de datos: File → Open Data Folder.
  3. Copie Dralvo_GoldMaster.ex5 en MQL5\Experts\. (O copie el .mq5 y pulse Compile (F7).)
  4. Fuente de datos CFTC — elija una: (a) en vivo vía API (recomendado) → configure UseLiveAPI=true + agregue la URL a la lista blanca (sección 2), sin necesidad de CSV; (b) por archivo → deje UseLiveAPI=false y debe copiar cftc_status.csv en MQL5\Files\ (si falta, aparece CFTC FILE NOT FOUND). El backtest siempre necesita el CSV.
  5. Vuelva a abrir MT5 → Navigator → Expert Advisors, clic derecho en Refresh.
  6. Abra un gráfico de XAUUSD, cambie al timeframe D1 y arrastre Dralvo GoldMaster al gráfico.
  7. Pestaña Common: marque Allow Algo Trading. Pestaña Inputs: ingrese su License key + MT5 Account (sección 2), cargue un .set.
  8. Pulse OK → active Algo Trading en la barra de herramientas (se enciende en verde).
Este es un sistema diario: las señales se evalúan una vez por cada nueva vela D1. Es normal pasar varios días sin operaciones — no espere entradas intradía.

2.Activación de licencia — OBLIGATORIA en demo/real

El EA tiene una verificación de licencia: en un gráfico real, llama al servidor de Dralvo para validar la clave. Sin este paso el EA NO funcionará y mostrará el aviso Contact telegram: @edgardinh86.

En el Strategy Tester (backtest) la verificación de licencia se omite — el backtest siempre se ejecuta sin clave.

Paso 1 — Obtenga su License key

Cómo obtener la clave depende de su plan:

PlanDónde obtener la License key
VIPInicie sesión en el Dashboard en https://www.dralvo.com → License / Key → genere una clave para cada cuenta MT5.
FreeContacte por Telegram @edgardinh86 para recibir una License key.
  • En cualquier plan, la clave está vinculada a un número de cuenta MT5 específico — debe indicar/registrar el número correcto.
  • Vea su número de cuenta en: Navigator, o Tools → Options → Server, o en el título de la ventana de MT5 (6–10 dígitos).

🔹 Plan VIP — sin cupos de cuenta: cada cuenta VIP recibe un número fijo de cupos. Cuando haya usado todos (máx.) y quiera añadir una nueva cuenta, contacte al administrador para obtener más.

Paso 2 — Lista blanca de WebRequest ⚠️ El paso más olvidado

  1. Abra Tools → Options (Ctrl+O).
  2. Vaya a la pestaña Expert Advisors.
  3. Marque ☑ Allow WebRequest for listed URL.
  4. Escriba exactamente: https://www.dralvo.com
  5. Pulse OK.
Esta misma URL también se usa para la API en vivo de CFTC cuando UseLiveAPI=true. Sin la lista blanca → la licencia falla y el panel muestra API: OFFLINE. Este es el motivo #1 por el que el EA no funciona aunque la clave esté correcta.

3.Cómo opera este EA

En cada nueva vela D1, el EA solo abre una compra (BUY) cuando todos los filtros se cumplen a la vez:

FiltroSignificado
CFTC alcistaEl posicionamiento de CFTC se inclina comprador. Fuente: en vivo + UseLiveAPI=true → directo desde la API; en otro caso (backtest, o UseLiveAPI=false) → leído de cftc_status.csv.
TRENDEMA50 > EMA200 en D1 (tendencia alcista).
PB (Pullback/Retroceso)El precio ha retrocedido lo suficiente respecto al máximo reciente (PullbackPct, PullbackBars).
HL (Higher-Low/Mínimo Ascendente)El mínimo de la vela más reciente está por encima del mínimo más bajo de hace ~20 velas (estructura alcista).

También requiere un cooldown (≥ 5 días desde la última operación, máximo 1 operación por día). Al entrar: SL = precio − 1.5×ATR, dimensionamiento por % de riesgo. Gestión: trailing TrailATR×ATR desde el máximo, cierre forzado tras MaxHoldDays.

El panel muestra 3 indicadores TREND / PB / HL (verde = aprobado, rojo = todavía no). La pestaña Experts/Journal imprime una línea ENTRY-CHK … en cada vela indicando exactamente qué filtro bloqueó la entrada.

6.Archivos .set y datos CFTC

4 setfiles según el perfil de riesgo (verificados — vea la sección 7):

SetfileRiesgo/operaciónPara
Dralvo_GoldMaster_SAFE.set1%Preservación de capital / cuentas pequeñas (DD ~6%).
Dralvo_GoldMaster_BALANCED.set2%Predeterminado recomendado (DD ~11%).
Dralvo_GoldMaster_AGGRESSIVE.set3%Crecimiento más rápido (DD ~15%).
Dralvo_GoldMaster_MAX.set5%Máxima ganancia, DD alto (~24%).
Archivo de soporteUbicaciónFunción
cftc_status.csvMQL5\Files\Datos diarios de CFTC. Obligatorio para backtests y para live con UseLiveAPI=false. No necesario en live con UseLiveAPI=true (usa la API).

Cargar un set: pestaña Inputs → Load → elija el archivo según su perfil. Guardar su propia configuración: Save.

Setfiles por tamaño de cuenta (selección rápida)

Para mantener el mismo patrón que los demás EA, también hay archivos Dralvo_GoldMaster_<PERFIL>_<tamaño>.set (24 archivos = 4 perfiles × 6 tamaños: 1k/5k/10k/20k/50k/100k). Solo elija el perfil + saldo correctos y cárguelo.

💡 Importante: GoldMaster calcula el lote totalmente según RiskPercent (% del saldo × distancia del SL), por lo que se ajusta automáticamente a cualquier tamaño de cuenta. Por eso, los archivos del mismo perfil (p.ej. todos los BALANCED_*) son idénticos — el tamaño en el nombre del archivo es solo para selección rápida y no cambia el funcionamiento del EA. Un solo archivo de perfil base sirve tanto para 1.000 como para 100.000 USD.
Si usa el archivo CSV (UseLiveAPI=false), debe actualizarse periódicamente para que "CFTC alcista" refleje la actualidad. Para evitar actualizaciones manuales → configure UseLiveAPI=true para que el EA lo obtenga vía API. InpLicenseKey/InpMT5Account son individuales y no se guardan en archivos .set compartidos.

7.Rendimiento del backtest (verificado)

Cifras obtenidas en XAUUSD D1, datos históricos reales de Exness (modelo 1-minute OHLC), período jul/2018 → jun/2026 (~8 años), apalancamiento 1:500, sin TP fijo (salida por trailing). Período completo: 94 operaciones, tasa de acierto 39.4%, Profit Factor ~2.4–2.6 — entradas selectivas, alta ganancia por operación.

Lectura rápida: el Profit Factor se mantiene estable en ~2.5 y el Max DD (equity) permanece por debajo del 24%, aumentando con el riesgo (1% ≈ 6%, 2% ≈ 11%, 3% ≈ 15%, 5% ≈ 24%). El nivel 2% ofrece un excelente equilibrio: +181% con solo ~11% de DD, Recovery Factor 5.3.

⚠️ Sistema de swing a largo plazo (~12 operaciones/año) — evalúe a lo largo de varios años, no semanal/mensual. Importante: los resultados dependen en gran medida de la integridad del historial de datos del bróker (un bróker con historial de oro incompleto genera menos operaciones y menor ganancia). Los backtests no garantizan el futuro; el spread/comisión/deslizamiento reales afectan el resultado.

9.Aviso de riesgo

  • Operar con margen implica alto riesgo. Ninguna configuración garantiza ganancias.
  • Siempre ejecute un demo / forward test el tiempo suficiente antes de usar dinero real.
  • Los archivos .set incluidos son un punto de partida; ajústelos a su bróker y a la volatilidad real.
  • La calidad de los datos CFTC afecta directamente las señales; el backtest no garantiza el futuro.
  • Opere solo con el capital que pueda permitirse perder.
