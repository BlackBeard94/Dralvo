1.Cài đặt

  1. Đóng MetaTrader 5 (nếu đang mở).
  2. Mở thư mục dữ liệu: File → Open Data Folder.
  3. Chép Dralvo_GoldMaster.ex5 vào MQL5\Experts\. (Hoặc chép .mq5 rồi Compile (F7).)
  4. Nguồn dữ liệu CFTC — chọn 1 trong 2: (a) live tự lấy qua API (khuyến nghị) → đặt UseLiveAPI=true + whitelist URL (mục 2), không cần file CSV; (b) dùng file → để UseLiveAPI=false thì phải chép cftc_status.csv vào MQL5\Files\ (thiếu là báo CFTC FILE NOT FOUND). Backtest luôn cần file CSV.
  5. Mở lại MT5 → Navigator → Expert Advisors, chuột phải Refresh.
  6. Mở chart XAUUSD, chuyển khung D1, kéo Dralvo GoldMaster vào chart.
  7. Tab Common: tick Allow Algo Trading. Tab Inputs: điền License key + MT5 Account (mục 2), nạp .set.
  8. Bấm OK → bật Algo Trading trên thanh công cụ (sáng xanh).
Đây là hệ Daily: tín hiệu chỉ được đánh giá một lần mỗi nến D1 mới. Bình thường nhiều ngày mới có 1 lệnh — đừng kỳ vọng vào lệnh trong ngày.

2.Kích hoạt License — BẮT BUỘC khi chạy demo/live

EA có cổng bản quyền: khi gắn vào chart thật, nó gọi máy chủ Dralvo để kiểm tra key. Thiếu bước này EA sẽ KHÔNG chạy và hiện cảnh báo Liên hệ telegram : @edgardinh86.

Trong Strategy Tester (backtest) cổng license được bỏ qua — backtest luôn chạy mà không cần key.

Bước 2 — Whitelist WebRequest ⚠️ Hay quên nhất

  1. Vào Tools → Options (Ctrl+O).
  2. Mở tab Expert Advisors.
  3. Tick ô ☑ Allow WebRequest for listed URL.
  4. Gõ chính xác: https://www.dralvo.com

Cùng URL này cũng dùng cho CFTC API trực tiếp khi bật UseLiveAPI=true. Không whitelist → license fail và dashboard hiện API: OFFLINE. Đây là nguyên nhân số 1 khiến điền key vẫn không chạy.

3.EA này giao dịch thế nào

Mỗi nến D1 mới, EA chỉ vào lệnh BUY khi đồng thời thỏa các bộ lọc:

Bộ lọcÝ nghĩa
CFTC bullishDữ liệu vị thế CFTC nghiêng mua. Nguồn: live + UseLiveAPI=true → lấy trực tiếp từ API; còn lại (backtest, hoặc UseLiveAPI=false) → đọc từ cftc_status.csv.
TRENDEMA50 > EMA200 trên D1 (xu hướng tăng).
PB (Pullback)Giá đã hồi xuống đủ sâu so với đỉnh gần nhất (PullbackPct, PullbackBars).
HL (Higher-Low)Đáy nến gần nhất cao hơn đáy thấp nhất ~20 nến trước (cấu trúc tăng).

Ngoài ra cần cooldown (cách lệnh trước ≥ 5 ngày, mỗi ngày tối đa 1 lệnh). Khi vào lệnh: SL = giá − 1.5×ATR, sizing theo % rủi ro. Quản lý: trailing TrailATR×ATR bám đỉnh, đóng bắt buộc sau MaxHoldDays.

Dashboard hiển thị 3 token TREND / PB / HL (xanh = đạt, đỏ = chưa đạt). Tab Experts/Journal in dòng ENTRY-CHK … mỗi nến cho biết chính xác cổng nào chặn.

File phụ trợVị tríVai trò

Nạp set: tab Inputs → Load → chọn file theo khẩu vị. Lưu cấu hình riêng: Save.

Bộ setfile theo mức vốn (chọn nhanh)

Để đồng bộ cách chọn với các EA khác, có thêm bộ file Dralvo_GoldMaster_<TIER>_<vốn>.set (24 file = 4 tier × 6 mức vốn: 1k/5k/10k/20k/50k/100k). Chỉ cần chọn đúng khẩu vị + số dư rồi nạp.

💡 Lưu ý quan trọng: GoldMaster tính lot hoàn toàn theo RiskPercent (% số dư × khoảng cách SL) nên tự co giãn theo mọi cỡ vốn. Vì vậy các file cùng tier (ví dụ mọi BALANCED_*) có nội dung giống hệt nhau — mức vốn trong tên file chỉ để chọn nhanh, không đổi cách EA chạy. Một file tier gốc đã đủ dùng cho cả 1.000 lẫn 100.000 USD.
Nếu dùng file CSV (UseLiveAPI=false), file cần được cập nhật định kỳ để CFTC bullish phản ánh đúng hiện tại. Muốn khỏi cập nhật tay → đặt UseLiveAPI=true để EA tự lấy qua API. InpLicenseKey/InpMT5Account gắn theo người dùng, không lưu trong .set chia sẻ.

7.Hiệu suất backtest (đã kiểm chứng)

Số liệu chạy trên XAUUSD khung D1, dữ liệu giá lịch sử thật của Exness (mô hình 1-phút OHLC), giai đoạn 07/2018 → 06/2026 (~8 năm), đòn bẩy 1:500, không TP cứng (thoát bằng trailing). Toàn giai đoạn: 94 lệnh, tỷ lệ thắng 39,4%, Profit Factor ~2,4–2,6 — vào lệnh chọn lọc, lãi mỗi lệnh lớn.

Đọc nhanh: Profit Factor ổn định ~2,5 và Max DD (equity) đều dưới 24%, tăng dần theo risk (1% ≈ 6%, 2% ≈ 11%, 3% ≈ 15%, 5% ≈ 24%). Mức 2% cân bằng rất tốt: +181% với DD chỉ ~11%, Recovery Factor 5,3.

✅ Xác minh ngày 2026-06-30 bằng MT5 Strategy Tester, Dralvo_GoldMaster v1.08 + các setfile theo mức rủi ro, trên dữ liệu Exness (XAUUSDm). Chi tiết 16 kịch bản: Backtest_Matrix_Exness.csv.
⚠️ Hệ swing dài hạn (~12 lệnh/năm) — đánh giá trên nhiều năm, không theo tuần/tháng. Quan trọng: kết quả phụ thuộc đáng kể vào độ đầy đủ dữ liệu lịch sử của broker (broker thiếu lịch sử vàng dài hạn sẽ ra ít lệnh & lợi nhuận thấp hơn). Backtest không đảm bảo tương lai; spread/commission/độ trượt thực tế đều ảnh hưởng.

9.Khuyến cáo rủi ro

  • Giao dịch ký quỹ có rủi ro cao. Không có cấu hình nào đảm bảo lợi nhuận.
  • Luôn chạy demo / forward test đủ lâu trước khi dùng tiền thật.
  • File .set đi kèm là điểm khởi đầu, cần hiệu chỉnh theo broker và biến động thực tế.
  • Chất lượng dữ liệu CFTC ảnh hưởng trực tiếp tới tín hiệu; backtest không đảm bảo tương lai.
  • Chỉ dùng số vốn bạn chấp nhận mất được.
