1.Instalasi

  1. Tutup MetaTrader 5 (jika sedang terbuka).
  2. Buka folder data: File → Open Data Folder.
  3. Salin Dralvo_GoldMaster.ex5 ke MQL5\Experts\. (Atau salin file .mq5 lalu klik Compile (F7).)
  4. Sumber data CFTC — pilih salah satu: (a) live via API (disarankan) → atur UseLiveAPI=true + whitelist URL (bagian 2), tidak perlu file CSV; (b) mode file → biarkan UseLiveAPI=false dan Anda wajib menyalin cftc_status.csv ke MQL5\Files\ (jika tidak ada akan muncul CFTC FILE NOT FOUND). Backtest selalu membutuhkan CSV.
  5. Buka kembali MT5 → Navigator → Expert Advisors, klik kanan lalu Refresh.
  6. Buka chart XAUUSD, ganti ke timeframe D1, dan seret Dralvo GoldMaster ke chart.
  7. Tab Common: centang Allow Algo Trading. Tab Inputs: masukkan License key + MT5 Account Anda (bagian 2), muat file .set.
  8. Klik OK → aktifkan Algo Trading di toolbar (menyala hijau).
Ini adalah sistem harian: sinyal dievaluasi sekali per candle D1 baru. Wajar jika beberapa hari tidak ada transaksi — jangan berharap entry intraday.

3.Cara Kerja EA Ini

Pada setiap candle D1 baru, EA hanya membuka posisi BUY ketika semua filter terpenuhi sekaligus:

FilterArti
CFTC bullishPosisi CFTC condong beli. Sumber: live + UseLiveAPI=true → langsung dari API; selain itu (backtest, atau UseLiveAPI=false) → dibaca dari cftc_status.csv.
TRENDEMA50 > EMA200 pada D1 (tren naik).
PB (Pullback)Harga sudah retrace cukup dalam dari puncak terbaru (PullbackPct, PullbackBars).
HL (Higher-Low)Low candle terbaru berada di atas low terendah dari ~20 candle sebelumnya (struktur bullish).

Juga memerlukan cooldown (≥ 5 hari sejak transaksi terakhir, maksimal 1 transaksi per hari). Saat entry: SL = harga − 1.5×ATR, ukuran posisi berdasarkan % risiko. Manajemen: trailing TrailATR×ATR dari puncak, penutupan paksa setelah MaxHoldDays.

Dashboard menampilkan 3 indikator TREND / PB / HL (hijau = terpenuhi, merah = belum). Tab Experts/Journal mencetak baris ENTRY-CHK … setiap candle yang menunjukkan filter mana yang menghalangi entry.

6.File .set & Data CFTC

4 setfile berdasarkan profil risiko (sudah diverifikasi — lihat bagian 7):

SetfileRisiko/transaksiUntuk
Dralvo_GoldMaster_SAFE.set1%Menjaga modal / akun kecil (DD ~6%).
Dralvo_GoldMaster_BALANCED.set2%Default yang disarankan (DD ~11%).
Dralvo_GoldMaster_AGGRESSIVE.set3%Pertumbuhan lebih cepat (DD ~15%).
Dralvo_GoldMaster_MAX.set5%Profit maksimal, DD tinggi (~24%).
File pendukungLokasiFungsi
cftc_status.csvMQL5\Files\Data CFTC harian. Wajib untuk backtest dan untuk live dengan UseLiveAPI=false. Tidak diperlukan pada live dengan UseLiveAPI=true (menggunakan API).

Memuat set: tab Inputs → Load → pilih file sesuai profil Anda. Menyimpan konfigurasi sendiri: Save.

Setfile berdasarkan ukuran akun (pilihan cepat)

Untuk menyamakan pola pemilihan dengan EA lainnya, tersedia juga file Dralvo_GoldMaster_<PROFIL>_<ukuran>.set (24 file = 4 profil × 6 ukuran: 1k/5k/10k/20k/50k/100k). Cukup pilih profil + saldo yang tepat lalu muat.

💡 Penting: GoldMaster menghitung lot sepenuhnya berdasarkan RiskPercent (% saldo × jarak SL), sehingga otomatis menyesuaikan diri dengan ukuran akun mana pun. Itu sebabnya file dengan profil yang sama (misalnya semua BALANCED_*) identik — ukuran dalam nama file hanya untuk pilihan cepat dan tidak mengubah cara kerja EA. Satu file profil dasar sudah cukup untuk 1.000 maupun 100.000 USD.
Jika menggunakan file CSV (UseLiveAPI=false), file harus diperbarui secara berkala agar "CFTC bullish" mencerminkan kondisi terkini. Untuk menghindari update manual → atur UseLiveAPI=true agar EA mengambil data via API. InpLicenseKey/InpMT5Account bersifat per-pengguna dan tidak tersimpan dalam file .set yang dibagikan.

7.Performa Backtest (Terverifikasi)

Angka dijalankan pada XAUUSD D1, data harga historis nyata dari Exness (model 1-minute OHLC), periode Jul 2018 → Jun 2026 (~8 tahun), leverage 1:500, tanpa TP tetap (keluar via trailing). Seluruh periode: 94 transaksi, win rate 39,4%, Profit Factor ~2,4–2,6 — entry selektif, profit besar per transaksi.

* Akun 1.000 USD berjalan pada lot minimum (0,01) sehingga risiko sebenarnya "terpaksa" lebih tinggi dari yang dikonfigurasi — persentasenya tidak dapat langsung dibandingkan dengan modal yang lebih besar. Modal mulai 5.000 USD ke atas baru mencerminkan sistem secara akurat.

Ringkasan cepat: Profit Factor stabil di ~2,5 dan Max DD (equity) tetap di bawah 24%, meningkat seiring risiko (1% ≈ 6%, 2% ≈ 11%, 3% ≈ 15%, 5% ≈ 24%). Tingkat 2% memberikan keseimbangan sangat baik: +181% dengan DD hanya ~11%, Recovery Factor 5,3.

✅ Diverifikasi pada 30-06-2026 dengan MT5 Strategy Tester, Dralvo_GoldMaster v1.08 + setfile berdasarkan risiko, pada data Exness (XAUUSDm). Detail 16 skenario: Backtest_Matrix_Exness.csv.
⚠️ Sistem swing jangka panjang (~12 transaksi/tahun) — evaluasi selama beberapa tahun, bukan mingguan/bulanan. Penting: hasil sangat bergantung pada kelengkapan data historis broker (broker dengan riwayat emas jangka panjang yang tidak lengkap akan menghasilkan lebih sedikit transaksi & profit lebih rendah). Backtest tidak menjamin masa depan; spread/komisi/slippage nyata semuanya berpengaruh.

9.Peringatan Risiko

  • Trading margin berisiko tinggi. Tidak ada konfigurasi yang menjamin profit.
  • Selalu jalankan demo / forward test cukup lama sebelum menggunakan uang sungguhan.
  • File .set yang disertakan adalah titik awal; sesuaikan dengan broker dan volatilitas nyata Anda.
  • Kualitas data CFTC secara langsung memengaruhi sinyal; backtest tidak menjamin masa depan.
  • Hanya gunakan modal yang siap Anda relakan jika hilang.
