1.Installation

  1. Close MetaTrader 5 (if open).
  2. Open the data folder: File → Open Data Folder.
  3. Copy Dralvo_Gold_Scalp.ex5 into MQL5\Experts\. (Or copy the .mq5 and Compile (F7).)
  4. Reopen MT5 → Navigator → Expert Advisors, right-click Refresh.
  5. Open an XAUUSD chart, M15 timeframe, and drag Dralvo Gold Scalp onto the chart.
  6. Common tab: tick Allow Algo Trading. Inputs tab: enter your License key + MT5 Account (section 2), load a .set.
  7. Click OK → enable Algo Trading on the toolbar (lights up green).
When it runs correctly, the Gold Scalp panel appears in the top-left of the chart (see section 5).

2.License activation — REQUIRED for demo/live

The EA has a license gate: on a real chart it calls the Dralvo server to verify the key. Without this step the EA will NOT run and shows the warning Contact telegram: @edgardinh86.

In the Strategy Tester (backtest) the license gate is skipped — backtests always run without a key.

Step 1 — Get your License key

How you get the key depends on your plan:

PlanWhere to get the License key
VIPSign in to the Dashboard at https://www.dralvo.com → License / Key → generate a key for each MT5 account.
FreeContact Telegram @edgardinh86 to receive a License key.
  • Whichever plan, the key is bound to a specific MT5 account number — you must provide/register the correct account number.
  • Find your account number: Navigator, or Tools → Options → Server, or the MT5 window title (6–10 digits).

🔹 VIP plan — out of account slots: each VIP account gets a set number of slots. Once you've used all (max) accounts and want to add a new one, contact the admin to be granted more.

Step 2 — Whitelist WebRequest ⚠️ Most-forgotten step

  1. Open Tools → Options (Ctrl+O).
  2. Go to the Expert Advisors tab.
  3. Tick ☑ Allow WebRequest for listed URL.
  4. Type exactly: https://www.dralvo.com
  5. Click OK.
If you skip this step, even with the correct key the EA errors and won't run (MT5 blocks the connection). This is the #1 reason.

3.How this EA trades

Step 1 — Regime detection: uses ADX + EMA + higher-timeframe (HTF) bias to classify RANGE / UPTREND / DOWNTREND / TRANSITION.

Step 2 — Pick an entry setup:

SetupWhenDefault
A. Breakout + RetestA genuine break of a level then a pullback to retest → trade the continuation with the trend.On
B. Fakeout + ReclaimA false break that closes back inside the range → trade the reversal.On
C. Range FadeFade both edges when the market is ranging.Off

Step 3 — "House-money" pyramiding: it only adds to the position when the current basket is already at breakeven/safe (risk-free), with decreasing volume, using one shared basket SL trailing the structure — it never lets a winning basket turn negative.

The dashboard shows Regime, HTF Bias, ADX, and Status telling you whether the EA is scanning / waiting for a retest / holding a basket / paused.

💡 How it works: the EA sizes lots by % risk (Use_Risk_Percent=true) so it auto-scales to your balance — the 4 tier files above work for any account size, the % return holds whether 1,000 or 100,000 USD.

Setfiles by account size (optional)

If you want a quick pick by balance or prefer a fixed lot instead of %, use the <TIER>_<size>.set set (24 files = 4 tiers × 6 sizes). Each keeps Use_Risk_Percent=true (runs exactly like the base tier file) but has a pre-set Fixed_Lot right for that size as a fallback — if you switch Use_Risk_Percent=false, the EA runs the correct fixed lot with no manual math.

9.Risk disclaimer

  • Margin trading is high-risk. No configuration guarantees profit.
  • Always run a demo / forward test long enough before using real money.
  • Pyramiding increases both profit and risk when misconfigured — understand Max_Adds, Add_Lot_Factor before raising them.
  • The bundled .set files are a starting point; tune them to your broker, spread and real volatility.
  • Only trade money you can afford to lose.
