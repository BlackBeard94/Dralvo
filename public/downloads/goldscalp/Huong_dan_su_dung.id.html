1.Instalasi

  1. Tutup MetaTrader 5 (jika sedang terbuka).
  2. Buka folder data: File → Open Data Folder.
  3. Salin Dralvo_Gold_Scalp.ex5 ke MQL5\Experts\. (Atau salin file .mq5 lalu klik Compile (F7).)
  4. Buka kembali MT5 → Navigator → Expert Advisors, klik kanan lalu Refresh.
  5. Buka chart XAUUSD, timeframe M15, dan seret Dralvo Gold Scalp ke chart.
  6. Tab Common: centang Allow Algo Trading. Tab Inputs: masukkan License key + MT5 Account Anda (bagian 2), muat file .set.
  7. Klik OK → aktifkan Algo Trading di toolbar (menyala hijau).
Jika berjalan dengan benar, panel Gold Scalp akan muncul di sudut kiri atas chart (lihat bagian 5).

3.Cara Kerja EA Ini

Langkah 1 — Deteksi Regime: menggunakan ADX + EMA + bias timeframe lebih tinggi (HTF) untuk mengklasifikasikan RANGING / TREN NAIK / TREN TURUN / TRANSISI.

Langkah 2 — Pilih setup entry:

SetupKapanDefault
A. Breakout + RetestBreakout asli diikuti pullback untuk retest level → tradingkan kelanjutan searah tren.Aktif
B. Fakeout + ReclaimBreakout palsu yang close kembali di dalam range → tradingkan reversal.Aktif
C. Range FadeFade kedua sisi saat pasar ranging.Nonaktif

Langkah 3 — Pyramiding "house money": hanya menambah posisi saat basket saat ini sudah breakeven/aman (risk-free), dengan volume menurun, menggunakan satu SL basket bersama yang trailing mengikuti struktur — tidak pernah membiarkan basket untung berbalik menjadi rugi.

Dashboard menampilkan Regime, Bias HTF, ADX, dan Status yang menunjukkan apakah EA sedang scanning / menunggu retest / menahan basket / dijeda.

💡 Cara kerja: EA menghitung lot berdasarkan % risiko (Use_Risk_Percent=true) sehingga otomatis menyesuaikan diri dengan saldo Anda — 4 file profil di atas berfungsi untuk ukuran akun mana pun, persentase return tetap sama baik untuk 1.000 maupun 100.000 USD.

Setfile berdasarkan ukuran akun (opsional)

Jika Anda ingin pilihan cepat berdasarkan saldo atau lebih suka lot tetap daripada %, gunakan kumpulan file <PROFIL>_<ukuran>.set (24 file = 4 profil × 6 ukuran). Setiap file tetap menggunakan Use_Risk_Percent=true (berjalan persis seperti file profil dasar), tetapi sudah memiliki Fixed_Lot yang telah diatur untuk ukuran tersebut sebagai cadangan — jika Anda beralih ke Use_Risk_Percent=false, EA akan berjalan dengan lot tetap yang benar tanpa perhitungan manual.

⚠️ Backtest tidak menjamin masa depan. Ini adalah sistem scalping yang membutuhkan broker dengan spread rendah; spread/komisi/slippage berpengaruh langsung → lakukan forward-test di demo terlebih dahulu. Timeframe lain (M5/M1/H1) yang diuji selalu berkinerja lebih buruk daripada M15.

9.Peringatan Risiko

  • Trading margin berisiko tinggi. Tidak ada konfigurasi yang menjamin profit.
  • Selalu jalankan demo / forward test cukup lama sebelum menggunakan uang sungguhan.
  • Pyramiding meningkatkan profit maupun risiko jika salah konfigurasi — pahami Max_Adds, Add_Lot_Factor sebelum menaikkannya.
  • File .set yang disertakan adalah titik awal; sesuaikan dengan broker, spread, dan volatilitas nyata Anda.
  • Hanya gunakan modal yang siap Anda relakan jika hilang.
