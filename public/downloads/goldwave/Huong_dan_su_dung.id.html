1Ikhtisar

Dralvo GoldWave adalah EA scalping XAUUSD (Emas) untuk MetaTrader 5, berjalan pada tick nyata. Idenya: strategi pemulihan (martingale balik arah) menghasilkan saat pasar menyamping dan mati saat tren kuat — maka GoldWave hanya masuk saat mendeteksi pasar SIDEWAYS di M1.

  • Simbol/Timeframe: XAUUSD, sinyal dihitung di M1.
  • Tipe: Scalp sideways + SAR-martingale (satu posisi dalam satu waktu).
  • Perisai: batas langkah (menahan ekor) + equity-guard (rem keras).

2Mekanisme masuk

  1. Deteksi sideways (M1): ADX < ambang (tanpa tren) + harga berada dalam range N candle. Di luar sideways → TIDAK masuk.
  2. Mean-reversion di tepi: harga menyentuh dasar range → BUY, menyentuh puncak → SELL. Lot = base.
  3. Pemulihan (SAR-martingale): jika kena SL → balik arah + kalikan lot (×2.0 atau ×2.75) dan masuk lagi seketika. Ulangi sampai menang.
  4. Menang → reset lot ke base, tunggu sinyal sideways baru.
  5. Mencapai batas langkah → PUTUS rantai (terima rugi, reset) — menahan risiko ekor.
Kenapa TANPA trailing / potong-tren? Sudah di-backtest: keduanya merusak matematika pemulihan martingale (order menang harus mencapai TP penuh untuk memulihkan rantai) → memperburuk hasil. Proteksi modal yang benar = batas langkah + equity-guard + modal cukup.

  1. Salin Dralvo_GoldWave.ex5 ke MQL5\Experts, refresh Navigator.

5Risiko & catatan

  • Ini martingale: kurva ekuitas terlihat bagus hampir selalu, tapi satu rantai rugi melewati batas = rugi besar / akun hangus. Batas + equity-guard mengurangi, bukan menghilangkan.
  • Lot ekor meledak eksponensial: makin tinggi pengali, makin brutal lot di langkah dalam (HighProfit langkah 10 = 90 lot). Modal HARUS menyerapnya.
  • Hanya jalan saat sideways, jadi ada masa EA "diam" menunggu range — wajar, bukan bug.
  • Broker spread rendah untuk Emas; di-backtest 34 bulan, tapi pasar ke depan bisa berbeda.

⚠️ Backtest masa lalu TIDAK menjamin masa depan. GoldWave adalah EA martingale — berisiko akun hangus saat tren kuat berjalan melewati batas. Modal harus menutup lot pada langkah ekor.
