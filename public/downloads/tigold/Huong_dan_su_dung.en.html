1.Installation

  1. Close MetaTrader 5 (if open).
  2. Open the data folder: File → Open Data Folder.
  3. Copy Dralvo TiGold.ex5 into MQL5\Experts\. (Or copy the .mq5 file, open MetaEditor and press Compile (F7).)
  4. Reopen MT5 → Navigator → Expert Advisors, right-click Refresh if you don't see it.
  5. Drag Dralvo TiGold onto an XAUUSD or BTCUSD chart (M1/M5 recommended).
  6. Common tab: tick Allow Algo Trading. Inputs tab: click Load and choose the right .set file.
  7. Click OK, then enable Algo Trading on the toolbar (lights up green).
When it runs correctly, the Dralvo TiGold panel appears in the top-left of the chart (see section 4).

2.Understanding "points" — read before tuning

Most parameters are measured in points, and the point value changes by broker:

SymbolDigits1 pointConversion
3-digit gold (e.g. XAUUSDm)30.0011.00 price = 1000 points
2-digit gold (e.g. XAUUSD)20.011.00 price = 100 points
2-digit BTC (e.g. BTCUSDm)20.011 USD = 100 points
The same point count gives different results across brokers. Always use the .set file matching your broker/symbol, or convert yourself. On startup the EA prints the symbol's digits and point to the Journal.

3.Parameter reference

General

ParameterDefaultMeaning
InpSymbol(empty)Trading symbol. Empty = use the chart's symbol.
InpMagic20260626Order identifier. Use a distinct number per EA/symbol to avoid conflicts.
InpSlippage30Max slippage on fill (points).
InpMaxSpread350Max spread allowed to enter (points). Wider → skip. 0 = off.

Entry Engine

ParameterDefaultMeaning
InpEntrySensitivity0Signal detection sensitivity (points). 0 = automatic.
InpEntryFilter3Confirmation steps before entering. Higher = more selective, fewer trades.
InpReentryDelay0Minimum wait between two trades (seconds).

Trend Filter

ParameterDefaultMeaning
InpUseTrendFiltertrueOnly enter when the trend is strong enough in the signal's direction.
InpTrendPeriod3Number of M1 candles used to measure the trend.
InpTrendStrength300Trend threshold (points). Larger = requires a stronger trend.

Time Filter

ParameterDefaultMeaning
InpUseTimeFiltertrueEnable trading-hours limits.
InpStartHour8Hour entries are allowed to start (server time, 0–23).
InpEndHour22End hour. Start = End → no limit.

Protection — capital protection

ParameterDefaultMeaning
InpMaxLossStreak0How many consecutive losses trigger a pause. 0 = off.
InpPauseMinutes60Pause duration after being triggered (minutes).
InpDailyStopPercent0Loss above this % of start-of-day equity pauses for the day (auto-resumes next day). 0 = off.

News Filter — LIVE only

ParameterDefaultMeaning
InpUseNewsFiltertrueStop entering around news times (from the MT5 Economic Calendar).
InpNewsMinsBefore15Minutes to pause before news.
InpNewsMinsAfter15Minutes to pause after news.
InpNewsImpactHigh onlyImpact to avoid: High only, or High + Medium.
InpNewsCurrenciesUSDCurrencies to watch (comma-separated, e.g. USD,EUR). Empty = all.
The news filter does not work in the Strategy Tester (the tester has no calendar). It only applies on demo/live.

Exit / Trailing

ParameterDefaultMeaning
InpStopLoss2500Initial stop loss (points).
InpTrailStart500Minimum profit to start moving the SL (points).
InpTrailDistance1000Distance the SL trails behind price (points).
InpTrailStep50Minimum step each time the SL is moved (points).

Money Management

ParameterDefaultMeaning
InpLotModeFixed lotLot mode: Fixed, or Auto (by capital).
InpFixedLot0.10[Fixed] Fixed volume.
InpEquityBasisBalance[Auto] Capital basis: Balance / Equity / Free margin.
InpCapitalPerLot1000[Auto] For every this much capital...
InpLotPerStep0.10[Auto] ...use this much lot.
InpMaxLot0Safety lot cap. 0 = broker limit only.
Auto-lot formula: lot = InpLotPerStep × (capital / InpCapitalPerLot).
Default: every 1000 capital → 0.10 lot ⇒ 5000 capital = 0.50 lot, 10000 capital = 1.00 lot. Lot rises with capital and falls with it.

Display

ParameterDefaultMeaning
InpShowPaneltrueShow the on-chart info panel.
InpDebugLogfalseDetailed logging (only enable for debugging).

5.Recommended .set files

FileFor
Dralvo TiGold.setXAUUSD (3-digit broker)
Dralvo TiGold BTC.setBTCUSD (2-digit)

Load: Inputs → Load → pick the file. Save your own config: Save.

If your gold broker is 2-digit, divide by 10 the point parameters in the gold set: InpStopLoss 2500→250, InpTrendStrength 300→30, InpTrailDistance 1000→100, InpMaxSpread 300→30, InpTrailStart 500→50, InpTrailStep 50→5.

6.Troubleshooting

SymptomCause / fix
EA doesn't take trades(1) Algo Trading off. (2) InpMaxSpread smaller than the broker spread → raise it. (3) Off the Time Filter hours. (4) Paused (see the Protection row). (5) InpTrendStrength too large for the symbol's range.
Too many / noisy tradesRaise InpEntryFilter and/or InpTrendStrength.
Lot too large/smallCheck InpLotMode, InpFixedLot or the Auto-lot formula.
SL/profit not as expectedParameters are in points — check the symbol's digits (section 2).
No dashboardInpShowPanel=true and running a real/visual chart.
News filter doesn't blockThe news filter is live-only, needs a broker with an Economic Calendar; no effect in the Strategy Tester.

See details in the Toolbox → Experts and Journal tabs.

7.Risk disclaimer

  • Margin trading is high-risk. No configuration guarantees profit.
  • Always run a demo / forward test long enough before using real money.
  • The bundled .set files are a starting point; tune them to your broker, spread and real volatility.
  • Backtest results do not guarantee future results; real spread/slippage may differ.
  • Only trade money you can afford to lose.
