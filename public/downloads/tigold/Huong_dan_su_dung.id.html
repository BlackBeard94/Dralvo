1.Instalasi

  1. Tutup MetaTrader 5 (jika sedang terbuka).
  2. Buka folder data: File → Open Data Folder.
  3. Salin Dralvo TiGold.ex5 ke MQL5\Experts\. (Atau salin file .mq5, buka MetaEditor lalu klik Compile (F7).)
  4. Buka kembali MT5 → Navigator → Expert Advisors, klik kanan lalu Refresh jika belum muncul.
  5. Seret Dralvo TiGold ke chart XAUUSD atau BTCUSD (disarankan M1/M5).
  6. Tab Common: centang Allow Algo Trading. Tab Inputs: klik Load lalu pilih file .set yang sesuai.
  7. Klik OK, lalu aktifkan tombol Algo Trading di toolbar (menyala hijau).
Jika berjalan dengan benar, panel Dralvo TiGold akan muncul di sudut kiri atas chart (lihat bagian 4).

2.Memahami "points" — baca sebelum menyesuaikan

Sebagian besar parameter diukur dalam points, dan nilai point berbeda tergantung broker:

SimbolDigit1 pointKonversi
Emas 3 digit (mis. XAUUSDm)30.0011.00 harga = 1000 points
Emas 2 digit (mis. XAUUSD)20.011.00 harga = 100 points
BTC 2 digit (mis. BTCUSDm)20.011 USD = 100 points
Jumlah points yang sama memberikan hasil berbeda antar broker. Selalu gunakan file .set yang sesuai dengan broker/simbol Anda, atau konversi sendiri. Saat startup, EA mencetak digits dan point simbol ke Journal.

5.File .set yang Disarankan

FileUntuk
Dralvo TiGold.setXAUUSD (broker 3 digit)
Dralvo TiGold BTC.setBTCUSD (2 digit)

Memuat: tab Inputs → Load → pilih file. Menyimpan konfigurasi sendiri: Save.

Jika broker emas Anda menggunakan 2 digit, bagi 10 parameter points pada set emas: InpStopLoss 2500→250, InpTrendStrength 300→30, InpTrailDistance 1000→100, InpMaxSpread 300→30, InpTrailStart 500→50, InpTrailStep 50→5.

6.Pemecahan Masalah

GejalaPenyebab / solusi
EA tidak melakukan transaksi(1) Algo Trading belum aktif. (2) InpMaxSpread lebih kecil dari spread broker → naikkan. (3) Di luar jam Filter Waktu. (4) Sedang dijeda (lihat baris Protection). (5) InpTrendStrength terlalu besar untuk rentang simbol.
Transaksi terlalu banyak / berisikNaikkan InpEntryFilter dan/atau InpTrendStrength.
Lot terlalu besar/kecilPeriksa InpLotMode, InpFixedLot atau rumus lot otomatis.
SL/profit tidak sesuai harapanParameter dalam points — periksa digits simbol (bagian 2).
Dashboard tidak munculInpShowPanel=true dan berjalan di chart nyata/visual.
Filter berita tidak memblokirFilter berita hanya berfungsi saat live dan memerlukan broker dengan Kalender Ekonomi; tidak berlaku di Strategy Tester.

Lihat detail di tab Toolbox → Experts dan Journal.

7.Peringatan Risiko

  • Trading margin berisiko tinggi. Tidak ada konfigurasi yang menjamin profit.
  • Selalu jalankan demo / forward test cukup lama sebelum menggunakan uang sungguhan.
  • File .set yang disertakan adalah titik awal; sesuaikan dengan broker, spread, dan volatilitas nyata Anda.
  • Hasil backtest tidak menjamin hasil masa depan; spread/slippage nyata dapat berbeda.
  • Hanya gunakan modal yang siap Anda relakan jika hilang.
